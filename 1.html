Hypertext Markup Language (HTML) is the standard markup language for create web pages and web applications
Web browsers receive HTML documents from a wet server or from local storage and render them into multimedia web paes.
HTML describs the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elemaents are the building blocks of HTML pages. With HTML constructs, images and other objects, sucj as interactive forms may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as hedings, paragraphs, lists, links quotes and other items. HTML elements are delineated by tags, wrtten using angle brackets.
